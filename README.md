# Alinex

Frontend project built with Gulp, Bootstrap, SCSS, jQuery, Owl Carousel, and SVG sprites.

## Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher recommended)
- npm (comes with Node.js)

## Install

```bash
npm install
```

## Development

Start the dev server with live reload on `http://localhost:3000`:

```bash
npm run dev
```

## Production Build

Build optimized assets into `dist/`:

```bash
npm run build
```

## Project Structure

```
src/
  assets/        — static assets copied as-is
  components/    — reusable HTML partials (header, footer)
  fonts/         — custom fonts (TB-64 KAZ family)
  img/           — images (PNG, SVG)
  js/            — JavaScript source files
  scss/          — SCSS stylesheets (entry: main.scss)
  svg/           — SVG icons (compiled into sprite.svg)
  index.html     — main page template
  rules.html     — rules page template
dist/            — compiled output (generated by build)
gulpfile.js      — Gulp task configuration
```

## Build Pipeline

| Task     | Description                                        |
| -------- | -------------------------------------------------- |
| html     | Compile HTML partials (`@@include`) and minify      |
| styles   | Compile SCSS, autoprefix, minify CSS                |
| scripts  | Bundle jQuery + Owl Carousel + Bootstrap + app JS   |
| images   | Copy images to dist                                 |
| svg      | Generate SVG sprite from individual icons           |
| fonts    | Copy fonts to dist                                  |
| assets   | Copy static assets to dist                          |
